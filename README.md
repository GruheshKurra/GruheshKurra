## About Me

I build AI systems end-to-end — from implementing attention mechanisms from scratch to deploying deepfake detectors that run entirely on-device in under 500ms. Currently building **Zynthetix Krew**, an autonomous multi-agent developer tool, while doing independent AI/ML research. Completed **B.Tech CSE at KL University with a 9.72 CGPA**, published **7 papers** across IEEE, IJNRD, and Zenodo. I write a blog where I implement ML algorithms from the math up — no black boxes. `sleep: insufficient`.
